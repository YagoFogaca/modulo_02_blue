Começando com o HTML

HTML - Linguagem de Marcação de Hipertexto html é a marcação de uma pagina web,ou seja, e a parte que define o significado, como o conteúdo e a estrutura dentro da página.

Todo link, imagem, texto, video é disposto na tela atravez do HTML. Portanto, o HTML pode ser difinido como a marção de midia e textos em página web.

DOM

O DOM é o responsavel por criar a herença entre as tag , parent e children. No DOM temos um especie de arvore, sendo o DOM o topo dessa arvore e abaixo dele temos todo nosso arquivo HTML, dividido entre parent e children. E atravez do DOM podemos manipular todas as tags HTML, atravez do CSS, dando vida a página e com o JS dando a interação da mesma.

  • parent == tag pai / superior
  • children == tag filha / inferior ou atributo de uma tag

Estrutura basica do HTML5

  1. <!DOCTYPE html>
    • Essa tag tem como objetivo dizer ao navegador qual versão do html a nossa página web está usando. A declaração como acima diz respeito ao HTML 5.
  2. <html lang="pt-br">
    • A tag onde onde todo nosso código HTML estará. Já o atributo lang="pt-br é linguagem da minha página web
  3. <head>
    • É o cabeçalho da pagina, onde está todas as configurações da página web, configurações não visiveis para o usuario. Escopo dentro do head sempre será as configurações, ele pertence ao html, por isso chamar ele de filho do html.
  4. <meta charset="UTF-8">
    • Tipo de dicionario, conjunto de caracteres para o teclado.
  5. <meta http-equiv="X-UA-Compatible" content="IE=edge">
    • Compatibilidade com o navegador edge, se for internet explore, será tratado da mesma forma que o ED.
  6. <meta name="viewport" content="width=device-width, initial-scale=1.0" >
    • Essa tag tem com os atributos name="viewport" content="width=device-width, initial-scale=1.0", objetivo definir a largura de pixel da página como a mesma que a largura de pixel do dispositivo em que é visualizada, bem como definir o nível de zoom inicial como padrão. Isso é importante porque no design web responsivo você muitas vezes deseja exibir conteúdo com base na largura do dispositivo.
  7. <title>
    • Essa tag tem como objetivo colocar um titulo na página, titulo que aparece na aba do navegador.
  8. <body>
    • É o corpo do site, onde inserimos imagens, textos, vídeos, mapas. Ou seja, tudo que é visível para o usuario final.

HTML semântico

  1. Tags semânticas.
  2. Tem como objetico deixar mais claro e legivel nosso código. Deixando acessivel a qualquer desenvolvedor.
  3. Descrever o significado do conteúdo presente nesses documentos através de tags semânticas.
